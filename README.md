# PublicHolidayTracker

Görsel Programlama dersi için hazırladığım C# konsol uygulaması.  
Uygulama, 2023–2025 yılları arasındaki resmi tatilleri API üzerinden çekiyor ve menü üzerinden:

- Yıla göre tatil listeleme
- Tarihe göre tatil arama (gg-aa)
- İsme göre tatil arama
- 3 yılın tüm tatillerini gösterme

işlemlerini yapıyor.

API: https://date.nager.at/api/v3/PublicHolidays/

Proje .NET SDK + VS Code ile yazıldı.
